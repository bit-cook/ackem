# 敏感能力 · Sensitive Capabilities

> **产品**：Ackem v1.0.0  
> **读者**：法务、高级用户、平台审核  
> **原则**：默认关闭、逐项授权、本地优先、可审计

---

## 1. 概述

本文档列出 Ackem 中可能触及用户隐私或系统安全的敏感能力。每项能力标明：**用途、数据类型、留存策略、关闭路径、当前实装状态**。

## 3. 详细说明

### 3.1 语音识别（STT）

- **实现**：通过 `voice-service/`（Python 进程）调用本地或云端 STT
- **数据流**：麦克风 → Ackem 主进程 → voice-service → 文本 → 主进程（处理后丢弃音频）
- **用户控制**：可随时停止监听，音频不离开本机（若使用本地 STT）

### 3.2 微信桥接

- **实现**：通过 `channels/weixin/` 模块与微信通信
- **数据流**：微信消息 → Ackem 处理 → 可选回复
- **隐私说明**：消息内容存储在本地 `weixin_*` SQLite 表中，不会自动上传
- **用户控制**：可在设置中完全关闭桥接，断开后本地留存的消息可手动删除

### 3.3 前台窗口检测

- **实现**：轮询当前前台窗口标题（频率受策略控制）
- **用途**：让伴侣感知用户活动上下文（如"你在写代码"），用于主动对话
- **数据存储**：`foreground_history` 表仅保留最近 N 条记录，自动裁剪
- **用户控制**：设置中关闭后停止轮询，已有记录可清除

### 3.4 扩展网络访问

- **实现**：扩展通过 `network_outbound` 权限声明所需访问的域名
- **授权模型**：安装时弹窗 → 用户批准 → 运行时按声明域名访问
- **审计**：扩展的网络请求记录在 `data/logs/`
- **用户控制**：可在扩展管理中撤销权限

## 4. 数据留存汇总

| 数据类别 | 位置 | 默认保留 | 用户可删 |
|----------|------|----------|----------|
| 聊天历史 | `data/companion/chat-history-*.json` | 无限制（UI 裁剪 2000 条/会话） | ✅ |
| 记忆事实 | `data/memory/facts/facts.v2.json` + SQLite | 永久（自动衰减/退休） | ✅ |
| 日记 | `data/diary/*.md` | 永久 | ✅ |
| 前台窗口历史 | `foreground_history` 表 | 最近 N 条 | ✅（设置中清除） |
| 微信消息 | `weixin_*` 表 | 永久 | ✅（设置中清除） |
| 日志 | `data/logs/` | 按大小滚动 | ✅ |
| 语音音频 | 内存临时 | 处理即弃 | — |
| API Key | `ackem-app-settings.json` | 永久 | ✅（设置中删除） |

## 5. 未实装能力

以下能力在本文档中预留但 **v1.0.0 尚未实装**：

- 浏览器历史访问
- 回收站/文件操作监控
- 屏幕截图分析
- GPS/地理位置
- 摄像头访问

这些能力若在未来版本实装，将遵循同样的默认关闭、逐项授权原则。
