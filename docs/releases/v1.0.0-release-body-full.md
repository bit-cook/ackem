## Ackem v1.0.0

首个对外开源版本 — Windows 绿色便携版（解压即用，无需安装 Node.js）。

### 下载 · Windows（官方）

| 包 | 说明 |
|----|------|
| **Ackem-1.0.0-win-x64.zip** | 绿色版（推荐），完整解压后双击 `Ackem.exe` |

### 使用前

1. **完整解压** zip 到 SSD 目录（勿在 zip 内直接运行）
2. 双击 `Ackem.exe` 或 `启动 Ackem.bat`
3. 首次启动约 10–30 秒（embedding 模型自动解压）
4. 在 **设置 → 模型与 API** 配置 LLM 接口

#### 安装（macOS Gatekeeper）

本构建未经 Apple 签名，首次打开可能提示「已损坏」——**属正常现象**。

1. 将 `Ackem.app` 拖入「应用程序」后，在终端执行：`xattr -cr /Applications/Ackem.app`
2. **右键** `Ackem.app` → **打开** → 在对话框中再次点 **打开**

#### v3 相对 v2 的修复

- 记忆引擎：修复 macOS 打包版 `resources/models` 路径解析
- 语音服务：修复 Python 3.9 类型语法兼容导致的启动超时
- 麦克风：新增 `entitlements.mac.plist` 音频输入权限

#### 已知限制

- 无 Apple 代码签名（社区构建）
- 前台检测 / 部分 Windows 专有 TTS 与媒体能力不可用或降级
- 内置更新通道仅识别 Windows `Ackem-*-win-x64.zip`
