# 贡献者公约 · 行为准则

## 我们的承诺

我们作为成员、贡献者和领导者，承诺让参与我们的社区成为每个人的无骚扰体验，不论年龄、体型、可见或不可见的残疾、种族、性别特征、性别认同与表达、经验水平、教育程度、社会经济地位、国籍、个人外貌、种族、宗教或性取向。

## 我们的标准

有助于营造积极环境的行为包括：
- 对他人表现出同理心和善意
- 尊重不同意见、观点和经历
- 给予并优雅地接受建设性反馈
- 承担责任，向受影响者道歉并从中学习
- 关注不仅对个人有利，也对整个社区有利的事情

不可接受的行为包括：
- 使用性化的语言或图像，以及任何形式的性关注或挑逗
- 挑衅、侮辱或贬低性评论，以及人身或政治攻击
- 公开或私下骚扰
- 未经明确许可发布他人的私人信息
- 在专业环境中可能被合理认为不合适的其他行为

## 执行责任

社区领导者负责明确和执行我们的行为标准，并将对任何他们认为不适当、威胁、冒犯或有害的行为采取适当且公正的纠正措施。

## 范围

本行为准则适用于所有社区空间，也适用于个人在公共场所正式代表社区的情况。

## 执行

可通过 **jasonliu_lyf_2005@qq.com** 向社区领导者报告滥用、骚扰或其他不可接受的行为。

所有投诉将得到及时和公平的审查与调查。

## 处理指南

社区领导者将根据以下社区影响指南确定违规行为的后果：

1. **纠正**：不适当的语言或非专业性行为 → 私下书面警告
2. **警告**：单次或系列违规 → 警告，指定时间内不得与相关人互动
3. **临时封禁**：严重违规 → 暂时禁止任何形式的公共互动
4. **永久封禁**：屡次违规或严重违规 → 永久禁止公开互动
